# BubblesSelection SDK

Available for iOS 11+.

## Installation

### CocoaPods


Create `Podfile` (with `pod init` for example) and add `pod 'BubblesSelection'`:

```
use_frameworks!

target 'AppName' do
    pod 'BubblesSelection'
end
```

Install pods:

```
$ pod install
```

Import the module:

#### Swift

```
import BubblesSelection
```

#### Objective-C:

```
@import BubblesSelection;
```
